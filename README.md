# Swag window manager

### Dependencies

```sh
# Build dependencies
sudo pacman -S base-devel xorg-server xorg-xinit xorg-xinput libinput libx11 libxinerama libxft

# Additional software
sudo pacman -S alacritty dex nitrogen picom

# Fonts
yay -S apple-fonts ttf-inconsolata-nerd
```

### Installation

```sh
cd swagwm
./install.sh
```
